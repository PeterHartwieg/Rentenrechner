## What to build

Register `/` in the public route registry from issue 02 and emit route-specific metadata. The homepage already renders three views (landing / compare / combine) via `detectSavedMode()`; the SSG-prerendered HTML must reflect the **landing variant** for crawlers, while compare/combine returning-user variants continue to hydrate client-side.

Add JSON-LD for `WebSite` + `Organization` + `WebApplication`, Open Graph + Twitter metadata, an internal-link hub to all topic pages, and a public brand audit (RentenWiki.de language, no new "Rentenrechner" copy).
