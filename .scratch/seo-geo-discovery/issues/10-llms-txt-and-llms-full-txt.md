**`llms.txt` format (per llmstxt.org spec):**
```
# RentenWiki.de

> Kostenloser Modellrechner für die Altersvorsorge in Deutschland. Keine Steuer-, Rechts- oder Anlageberatung. Quelloffen, nicht-kommerziell (PolyForm Noncommercial 1.0.0). Stand 2026.

## Rechner und Themen

- [<title>](<absolute-canonical-url>): <summary one-liner>
  (one entry per in-sitemap route, in registry source order, excluding /404)
```

**`llms-full.txt` format:**
```
# <title>

URL: <absolute-canonical-url>
Stand: <dateModified>

<stripped-text-of-prerendered-HTML>

---
```
(Repeated for each in-sitemap non-404 route, separated by `---`.)

**`generateLlmsTxt()` in `src/seo/llmsTxt.ts`:**
- Pure function, no side effects.
- Takes optional registry override (for testing).
- Outputs the `llms.txt` string.
- Unit tests pin the header, per-route lines, exclusion of `/404`, and exclusion of share-state routes.

**`llms-full.txt` generation:**
- Runs in `scripts/prerender.mjs` after all SSG passes.
- For each in-sitemap, non-404 route: read `dist/${slug}/index.html` (or `dist/index.html` for `/`), strip tags, trim.
- Write concatenated result to `dist/llms-full.txt`.
